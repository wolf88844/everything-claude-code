## Execution Constraint

Every day in this matrix is a candidate publishing slot, not an auto-publish instruction.

Daily execution must still follow the full workflow:
- topic confirmation
- Success Spec confirmation
- wechat-growth-workflow Step 1 to Step 3
- write / revise / drop decision
- drafting
- Editorial Pass with full revised body
- title pack
- visual packaging
- optional Xiaohongshu adaptation
- topic system write-back

If a topic fails the workflow check on that day, do not publish it just because it was scheduled. Replace or defer it.

- do not repeat already-written flagship topics as-is
- use hot signals only as translations into the same IP territory
- treat daily publishing as a mix of heavy posts, bridge posts, hot-topic translations, and lighter cards
- every topic carries a Success Spec
