# Daily Execution Checklist

Use this checklist when running one day's content strictly through the workflow.

## Before Writing
1. confirm today's scheduled topic
2. confirm the article role
3. confirm the primary goal
4. confirm the expected level
5. confirm minimum success signal

## Before Drafting
1. run market research
2. run deep research
3. run content system design
4. lock writing principles
5. confirm the piece is still worth writing

## Before Publishing
1. complete editorial pass
2. output full revised article body
3. generate titles
4. generate visual packaging
5. if needed, generate Xiaohongshu version

## After Publishing
1. check whether the main goal was achieved
2. note the minimum success signal
3. capture reusable lines, mechanisms, and sequel angles
4. update matrix or topic status when relevant
