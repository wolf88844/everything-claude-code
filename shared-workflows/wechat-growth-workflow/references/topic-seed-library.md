# Topic Seed Library

This file stores approved or promising topic directions for the user's公众号账号.

## Topic Status Legend

Use these labels when updating the library:
- `待写` — approved seed, not drafted yet
- `草稿` — drafted in workflow, not yet published or validated
- `已写` — written and considered complete
- `已验证` — published and supported by real feedback or data
- `已重写` — rewritten from a weaker or older angle

Whenever a topic is drafted or published, map it back here with:
- `Status`
- `Latest title`
- `Notes`

## Content Pillars

### Pillar A: 成长卡点拆解
- 面向有工作焦虑、停滞感、反复重启感的人
- 优先做高共鸣、高转发、高收藏的入口型内容

### Pillar B: 认知升级与自我管理
- 面向想从焦虑回到稳定主动性的人
- 适合做系统、时间、精力、迁移能力相关内容

### Pillar C: 技术职场判断力
- 面向 35+ 技术人、技术管理边缘人群、职业转型人群
- 适合建立专业判断感和差异化

### Pillar D: 经验变内容 / 个人品牌起步
- 面向想做副业、个人品牌、内容输出的人
- 适合承接到表达力、经验资产化、第二增长曲线
