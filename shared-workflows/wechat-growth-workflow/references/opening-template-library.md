# Opening Template Library

Use these opening templates to start公众号文章 in the user's preferred style.

## Core Rule

The opening should not begin by teaching.
It should begin by making the reader feel recognized.

## Template 1: Repeated Behavior

Use when the reader repeats a habit that feels familiar and slightly painful.

Pattern:
- name the behavior
- show how often it happens
- show the emotional comfort it creates
- reveal that the real problem is elsewhere

Example skeleton:
很多人其实很喜欢{某种行为}。

{某个固定时间点}一次。
{另一个固定时间点}再来一次。
{状态不好时}，还会给自己开一个新阶段。

{行为}一出现，人会立刻有一种{情绪收益}。

可问题是，很多人真正缺的，可能根本不是{表面答案}。
而是{更深答案}。

## Template 2: Wrong Explanation

Use when the article aims to overturn a common self-judgment.

Pattern:
- state the common explanation
- question it
- give the hidden explanation

Example skeleton:
很多人以为，自己一直卡住，是因为{常见解释}。

这是一个很常见的误会。

更常见的情况不是{A}。
而是{B}。

## Template 3: Surface vs Deep

Use when the article needs a calm but sharp diagnosis.

Pattern:
- describe how it looks on the surface
- describe what is happening underneath

Example skeleton:
表面上看，很多人一直都挺{表面评价}。
经常{行为1}，经常{行为2}，经常{行为3}。

但往深里看，问题不在{表面结论}。
而在{深层结构}。

## Template 4: Exhaustion Recognition

Use when the reader is tired, self-blaming, and wants relief.

Pattern:
- name the exhaustion
- remove shame
- redirect blame from character to structure

Example skeleton:
如果你这几年一直有一种感觉：
自己并不是不想变好，
只是总在断、总在掉、总像重新来过。

那你缺的，可能不是更狠一点。
而是一个能把你接住的{系统/结构/方法}。

## Template 5: Quiet Contradiction

Use when you want a more restrained opening.

Pattern:
- offer a soft contradiction
- expand it gradually

Example skeleton:
很多人并不是没有目标。
相反，他的问题往往是目标太多。

真正缺的，不是新的方向。
而是一套能把今天留给明天的结构。

## Opening Checklist

A strong opening should do at least four of these:
- create instant recognition
- contain at least one line worth underlining
- avoid abstract preaching
- avoid jumping into advice too early
- suggest the article's hidden thesis without fully explaining it
- make the reader want to keep going

## Banned Openings

Avoid openings like:
- "今天想和大家聊聊..."
- "在这个快节奏的时代..."
- "很多人都知道目标很重要..."
- any opening that sounds like a lecture before recognition happens
